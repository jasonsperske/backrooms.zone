BULK IMPORT (JSON)

Paste a JSON array of door objects. Each object maps to a door entry:

[ { "game_door_id": "door-west-0-7", "label": "West Exit", "dest_url": "https://..." }, { "game_door_id": "door-east-0-3" } ]
HOW TO USE IN YOUR GAME

When the player collides with a door, redirect them to:

/door?source=YOUR_ROOM_URL&door=DOOR_ID&x=...&y=...&z=...

The destination game receives ?bz_door=DOOR_ID and optional bz_x/y/z return-position coordinates.
